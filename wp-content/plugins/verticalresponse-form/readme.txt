== Description ==

**The VerticalResponse Form by ContactUs.com is a plugin to create and publish dynamic newsletter signup form experiences used by VerticalResponse subscriber to build their email marketing list.**

[vimeo http://vimeo.com/76663015]

The VerticalResponse Form by ContactUs.com allows plugin users to create newsletter signup forms from a library of beautifully-designed templates, which are easily-integrated to an VerticalResponse account. This signup form is used by businesses and websites who are VerticalResponse users to add new email subscribers from their WordPress website. 

VerticalResponse [www.VerticalResponse.com](http://www.verticalresponse.com) is email marketing software used by over 200,000 businesses to create, send and track email newsletters campaigns. The VerticalResponse Form plugin by ContactUs.com is designed to help VerticalResponse emails to use their websites to generate signups to their newsletter list.

== Installation ==

There are two ways to download and install the VerticalResponse Form by ContactUs.com:

**Downloading from WordPress Admin Panel (recommended)**

1. Log into your WordPress.org account as an administrator.
2. Go to the “Plugins” section from the side panel, and choose “Add New”.
3. Search for “VerticalResponse Form by ContactUs.com” to find the plugin, and choose to install.
4. WordPress will automatically install the plugin into your account.
5. Once completed, go to your list of plugins (bottom of your side panel), and choose to “Activate”.
6. After the plugin is activated, open the VerticalResponse plugin.
7. Log into your ContactUs.com account from inside the plugin to connect accounts.
8. Find your VerticalResponse API Key. (To generate your API Key, you need to log in to your VerticalResponse.com account. Click on “Account” => “Settings” => “API”, and you can retrieve your API URL and API Key from there.)
9. Complete the sign-up instructions to allow access and connect accounts, and then you’re done!


**Downloading from WordPress Plugin Directory.** 

1. Go to WordPress Plugin Directory (http://wordpress.org/plugins/).
2. Search for “VerticalResponse Form by ContactUs.com”.
3. After you found the Plugin, click on Download. The file should start downloading as .zip
4. After it’s downloaded, log into your WordPress.org account as an administrator.
5. Go to the “Plugins” section from the side panel, and choose “Add New”.
6. Click on “Upload” and select the downloaded file from your computer. It should be in the “Downloads” folder.
7. After it has uploaded, click on Install. WordPress will automatically install the plugin into your account.
8. Once completed, go to your list of plugins (bottom left of the panel), and click on “Activate”.
9. After the plugin is activated, open the VerticalResponse Plugin by ContactUs.com.
10. Log into your VerticalResponse.com account from inside the plugin.
11. Find your VerticalResponse API Key. (To generate your API Key, you need to log in to your VerticalResponse.com account. Click on “Account” => “Settings” => “API”, and you can retrieve your API URL and API Key from there.)
12. Complete the sign-up instructions to allow access and connect accounts, and then you’re done!

Detailed instructions, which screenshots, on how to implement the ContactUs.com Wordpress can be found <a href="http://help.contactus.com/entries/23229688-Adding-the-ContactUs-com-Plugin-for-WordPress" target="_blank">here</a>.

== Frequently Asked Questions ==

1. How does this Plugin help my website? ContactUs.com VerticalResponse plugin allows you to obtain newsletter subscribers in an easier and more efficient way. With our multiple templates, our plugin allows you to approach potential newsletter subscribers in a non-intrusive way. Our templates designs have been worked carefully, so people can easily find them and think of them as an opportunity to stay in touch with you.
2. What kind of implementation do you offer? Similar to our contact form, the VerticalResponse Plugin by ContactUs.com, offers to implement it as a proactive tab or embedded on a page you choose. It also allows you to customize the implementation, so you can decide what to have, on each page of your WordPress website.
3. How many subscribers can I get through this plugin? How many newsletter subscribers you sign-up will depend on how many visitors you have to your website. Our goal with this plugin is to generate more sign-ups in proportion to your overall web traffic. The plugin achieves this by offering more proactive form placements and using better-designed form templates.
4. Can I use both ContactUs.com Contact Form Plugin and ContactUs.com VerticalResponse Plugin at the same time? Currently it’s only possible to use one plugin at the time. We are working so it can be possible to use both plugins on your site at the same time. We will notify you when that goes live.
5. Where do I get my API Key? To generate your API Key, you need to log in to your VerticalResponse.com account. Click on “Account” => “Settings” => “API”, and you can retrieve your API URL and API Key from there.
