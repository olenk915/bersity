Add your own css class on the button. Set your own message request for each button.

== Description ==

[FACEBOOK AWD](http://facebook-awd.ahwebdev.fr "")

This plugin will add shortcode and widget that add app requests button.
If a user click on the button, Facebook will show a pop up with the choice list of friends to invite.
Add your own css class on the button. Set your own message request for each button.

[FACEBOOK AWD](http://facebook-awd.ahwebdev.fr "") v1.4 required.
